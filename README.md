# VIGIL Project Page

Static GitHub Pages site for:

**Staying VIGILant: Mitigating Visual Laziness via Counterfactual Visual Alignment in MLLMs**

The site is designed as a lightweight academic project page with:

- title, authors, affiliations, and project links
- teaser figure
- abstract and three main claims
- method overview
- result highlights
- BibTeX block

## Local Preview

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Deployment

Push this repository to GitHub and enable GitHub Pages from the `main` branch root.
